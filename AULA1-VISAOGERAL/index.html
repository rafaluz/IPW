Conceitos iniciais

Fala pessoal, aqui é o professor Rafael, na aula de hoje vamos começar a aprender HTML. Iremos começar aprendendo as tags básicas que são muito utilizadas durante o desenvolvimento web, como a tag p (parágrafo), h1 ... h6 que são os títulos.

Também iremos aprender tags de texto como a tag strong, que indica um texto importante. Tag b que deixa o texto em negrito, tag i que deixa o texto em italico, tag em que é texto enfatizado e muito mais.

Seguindo nessa linha, iremos aprender como criar links, com a tag a, como inserir imagens com a tag img. O mais legal de tudo é que iremos aprender a usar o CSS ao mesmo tempo que aprendemos HTML, tudo de uma vez só, sem complicações e sem estresse. blz? Ah, caso você queira se aprofundar, na w3schools tem tudo que você precisa sobre HTML.

Foco, força, fé e vamos que vamos meus amigos!
